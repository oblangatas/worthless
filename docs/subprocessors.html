Third-party infrastructure for wless.io

Worthless is self-hosted, open-source software. The Operator does not run a hosted service and does not receive, store, or process your API keys, prompt content, or response content. There is no customer-data pipeline, and therefore no subprocessor that handles your provider keys or model traffic.

The wless.io website and the Operator's contact email rely on a small amount of standard third-party infrastructure that processes limited operational data, as described in the Privacy Policy:

ServicePurposeData involved
Static website host (GitHub Pages) Serving the wless.io website Standard server-log information (IP address, user agent, requested URL, timestamp)
GitHub Source code, issues, and pull requests Whatever GitHub collects from visitors and contributors, under GitHub's own privacy policy
Email provider Receiving mail sent to wless.io addresses (legal@, privacy@, security@) The contents of email you choose to send

The Software itself runs entirely on systems you control and connects directly to the LLM providers whose API keys you own. The Operator is not a subprocessor of that traffic.

If the Operator adds analytics or other processors in the future, the change will be disclosed in the Privacy Policy at that time. For questions, contact privacy@wless.io.
